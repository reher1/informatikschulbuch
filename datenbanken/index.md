# Datenbanken

Das erste Thema in diesem Halbjahr ist die professionelle Speicherung von Daten. Heutzutage setzen nahezu alle Firmen auf Datenbanken für die verschiedensten Zwecke, so dass dieses Wissen zur wesentlichen Grundlage geworden ist, das ihr in quasi allen Berufen brauchen könnt. 

Wir beschränken uns im Grundkurs auf die "[relationalen Datenbanken](https://www.bigdata-insider.de/was-ist-eine-relationale-datenbank-a-643028/)". In diesen werden die Informationen in Tabellen gespeichert, die untereinander über Beziehungen (Relationen) verknüpft sind. 

> Ein **Datenbanksystem** ist eine systematische und strukturierte Zusammenfassung von Daten eines Problembereiches (**Datenbasis**) einschließlich der zur Eingabe, Verwaltung, Auswertung und Ausgabe erforderlichen Software (**Datenbankmanagementsystem**, DBMS).
**Merke: Datenbanksystem = Datenbasis + DBMS**
